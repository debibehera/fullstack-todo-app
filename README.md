# fullstack-todo-app
# ✅ Full-Stack Todo App

A full-featured Todo List Application built with **React**, **Node.js**, **Express**, and **MongoDB (Atlas)** with **JWT Authentication**, **CRUD operations**, and **real-time updates**.

---

## 🚀 Features

- 🔐 Authentication (JWT-based login/signup)
- ➕ Add Todos
- ✅ Mark as Completed
- ✏️ Edit Todos
- 🗑️ Delete Todos
- 🔁 Real-time UI Updates
- 🌐 Fully Connected Frontend and Backend
- 🌩️ MongoDB Atlas Integration
- 🧱 Clean Folder Structure (MVC + Context API)
- 🔒 Rate Limiting to prevent abuse
- 🌙 Dark mode (optional)
- 📦 Deployed on Render / Vercel

---

## 🧑‍💻 Technologies Used

### Frontend:
- React (with Vite)
- Tailwind CSS
- Context API
- Axios
- React Router

### Backend:
- Node.js
- Express
- MongoDB Atlas
- Mongoose
- Bcryptjs
- JsonWebToken
- Express Rate Limit
- Dotenv
